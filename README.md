# photobooth
## Prerequisites
! Add a config file named photobooth-config.yml to the top level of the project. Format:

```
flickr:
  url: 
  api_key: 

one:
  host: 
  shoutout_wall: 
  api_path_share: 
  api_path_upload: 
  user:
  pw: 
```

! Add a folder "photos" to the top level.

## Flickr API
* https://www.flickr.com/services/api/misc.urls.html
